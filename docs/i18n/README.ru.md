# OpenWrt Smart Reboot (с LuCI)

- Английская версия: [README.en.md](../../README.en.md)
- Индекс языков: [README.i18n.md](README.i18n.md)

Этот пакет перезагружает роутер в заданное раннее время только когда сеть считается неактивной.

## Компоненты

- `smart-reboot`: backend-служба проверки простоя и перезагрузки
- `luci-app-smart-reboot`: страница настройки LuCI

## Как это работает

1. Запускает `smart-reboot-check` в заданное время (по умолчанию: `04:00`)
2. Измеряет `rx_bytes + tx_bytes` на выбранных интерфейсах в течение `sample_seconds`
3. Если прирост ≤ `byte_threshold`, считается простоем и выполняется перезагрузка

## Значения по умолчанию

- Включено: `1`
- Время: `04:00`
- Выбрать все интерфейсы: `0`
- Интерфейс: `wan`
- Время выборки: `120`
- Порог простоя: `262144` байт (256KB)

## Параметры LuCI

- Время перезагрузки: `HH:MM`
- Выбор всех интерфейсов
- Контролируемые интерфейсы (логическая метка + имя устройства)
- Время последней автоперезагрузки
