# OpenWrt Smart Reboot (з LuCI)

- Англійська версія: [README.en.md](../../README.en.md)
- Індекс мов: [README.i18n.md](README.i18n.md)
- Гайд з upstream OpenWrt: [OPENWRT_UPSTREAM_SUBMISSION.md](../../OPENWRT_UPSTREAM_SUBMISSION.md)

Цей пакет перезавантажує роутер у заданий ранковий час лише коли мережа вважається неактивною.

## Компоненти

- `smart-reboot`: backend-служба перевірки простою та перезавантаження
- `luci-app-smart-reboot`: сторінка налаштування LuCI

## Як це працює

1. Запускає `smart-reboot-check` у вказаний час (типово: `04:00`)
2. Вимірює `rx_bytes + tx_bytes` на обраних інтерфейсах протягом `sample_seconds`
3. Якщо приріст ≤ `byte_threshold`, вважається простоєм і виконується перезавантаження

## Типові значення

- Увімкнено: `1`
- Час: `04:00`
- Вибрати всі інтерфейси: `0`
- Інтерфейс: `wan`
- Тривалість вибірки: `120`
- Поріг простою: `262144` байт (256KB)

## Параметри LuCI

- Час перезавантаження: `HH:MM`
- Вибір усіх інтерфейсів
- Інтерфейси моніторингу (логічна мітка + назва пристрою)
- Час останнього автоперезавантаження
